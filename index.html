Projects

SmArt Time

A time management website built by me using the Flask framework. The website stores the list of tasks that the user must complete along with the tasks associated deadline. It provides the user its recommendations regarding how much time he/she should spend on each task per day in order to complete it on time based on each task's deadline, priority and the total number of hours a user intends to work per day. Other notable features of the website include an implementation of a pomodoro time management technique and a productivity progress report provided based on user feedback.
